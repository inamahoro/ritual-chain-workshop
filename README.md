# MultiSigAIBounty – Multi-Signature Verification

This contract uses **multi-signature verification** where multiple parties must approve the winner before the reward is released.

## How it works
1. **Commit phase**: Participants submit hashed answers
2. **Reveal phase**: Participants reveal their answers
3. **AI scoring**: Owner sets AI quality scores
4. **Propose winner**: Owner proposes a winner
5. **Multi-sig approval**: Signers approve the winner
6. **Finalization**: Reward released when threshold met

## Why multi-sig?
Decentralized control with no single point of failure.
